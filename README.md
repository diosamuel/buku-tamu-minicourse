# flask-guest-book

## Overview

This is a guest book wrapped into a Webapp using [Flask](http://flask.pocoo.org/) Micro Framework. The app is connected to a MySQL database hosted for free in [remotemysql](https://remotemysql.com/).

## Dependencies

```sudo pip install -r requirements.txt```

## Usage

Once dependencies are installed, just run this to see it in your browser. 

```python guestbook.py```
